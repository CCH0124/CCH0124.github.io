Table of content

Basic command

  1. 查看 Docker 訊息
docker info
  1. 登入 hub.docker.com
docker login

Image command

  1. 查看本地端以下載的 image
docker images
  1. 移除 image
docker rmi  -f {image ID} # -f 強制，直接將運行中的 container 刪除
  1. 取得 container 的資訊
docker inspect {ImageID}
  1. 取得 image 的歷史紀錄
docker history  {ImageName}

Containers Command

  1. 取得運行的 container
docker ps -a
  1. 運行 container
docker run -it {imageName}
  1. 取得該 container 的 log
docker logs {CONTAINER}
  1. 啟動 container
docker start {ConatainerName}
  1. 停止 container
docker stop {ConatainerName}
  1. 暫停 container
docker pause  <ConatainerName>
  1. 取消暫停 container
docker unpause  {ConatainerName}
  1. kill 運行中的 container
docker kill {ConatainerName}
  1. 移除 container
docker rm {ConatainerName}
  1. 進入 container
docker exec -it {ConatainerName}
